# Python Basics for Data Engineering

This repository contains core Python scripts focused on
Data Engineering fundamentals such as file handling,
JSON parsing, reusable functions, and error handling.

## Structure
- csv_reader.py → Reads and filters CSV files
- json_parser.py → Parses JSON data
- functions.py → Reusable business logic
- error_handling.py → Robust exception handling
- venv_notes.md → Virtual environment setup

## How to Run
python csv_reader.py

## Skills Covered
- File I/O
- Data parsing
- Functions
- Error handling
- Virtual environments
